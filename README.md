# LearnReact

## React 기본기 학습
[React] (https://ko.react.dev/learn)

### 설치 
npm create vite@latest my-app -- --template react


#### 설정과정
Project name: ... my-app
Select a framework: » React
Select a variant: » JavaScript

#### 의존성 설치
cd my-app
npm install

#### 로컬 서버 실행
npm run dev

스타일링 : npm install sass
